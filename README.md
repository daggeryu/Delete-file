## 文件删除小工具

### 使用Swing，JDk1.8。适用于mac/win/Linux  
编写初衷是使用mac会生成一些不需要的隐藏文件，在win下回看到，强迫症受不了，还有下载一些压缩包解压出来包含一些.url文件，解压太多懒得一个一个去删除。所以为了投案只做了此小工具。  
效果如图，需要安装JDK，可双击打开，使用在app.jar目录此命令运行“java -jar app.jar” 可以看到删除的文件及目录。
